# CMatrix

The project aims to build a website for matrix operatins.

Following matrix operation will be done on the matrices:
- Adddition (2 matrices)
- Subtraction (2 matrices)
- Multiplication (2 matrices)
- Inverse (1 matrix)
- Adjoint (1 matrix)
- Transpose (1 matrix)
- Determinant (1 matrix)
- Trace(1 matrix)
